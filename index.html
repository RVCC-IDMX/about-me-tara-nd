We may make use of the background-image property without any further configuration. The url() function must be used to identify the picture source, regardless of whatever property we use. The value returned by the url() method will serve as a link to the background image, and the same concepts that apply to creating hyperlink pathways will apply here as well. Keep an eye out for different directories and be sure to provide the exact location of the photograph. The route will be included in parentheses and it will be quoted. As an example,: div { background-image: url("alert.png"); } Because the background image will, by default, repeat horizontally and vertically from the top left of the given element to fill up the element's background, adding a background picture alone with a url value may result in adverse outcomes. Fortunately, we have the ability to customize how and whether the image repeats by using the background-repeat and background-position properties. Background Repeat Unless otherwise specified, a background image will loop indefinitely, both vertically and horizontally. div { background-image: url("alert.png"); background-repeat: no-repeat; } The background-repeat property accepts four different values: repeat, repeat-x, repeat-y, and no-repeat Background Position Background pictures are usually placed in the left upper corner of an element. We can, however, control where the background picture is positioned in relation to that corner by using the background-position property. Background-position needs two values: a horizontal offset (the first value) and a vertical offset (the second value) (the second value). If just one number is supplied, it is utilized for the horizontal offset, and the vertical offset is set to 50% by default. Background images are often placed at the left-most corner of an element's left-most corner. The background-position attribute, on the other hand, allows us to specify where the background image should be placed in reference to the corner in question. If just one number is provided, it is used for the horizontal offset, and the vertical offset is set to 50% by default. If more than one number is provided, it is used for both the horizontal and vertical offsets. Because we're moving the background image away from the element's left top corner, the length values for that corner will be unique to that corner alone. To get the background-position value, we may use the keywords top, right, bottom, left, and center, as well as pixels, percentages, or any other measurement of length. Using keywords in conjunction with percentages has a very similar effect. Using the keyword value left top, which corresponds to the percentage value 0 0, you may maintain an image positioned in the element's left top corner.
